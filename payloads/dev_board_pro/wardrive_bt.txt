# wardrive_bt -- Bluetooth wardrive, logging BLE/BT devices to the SD card
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# `btwardrive -c` logs every detection (does not de-dupe repeats). Files are prefixed
# `bt_wardrive_` on the SD card. GPS coords require the GPS DIP switch routed to the
# CLI ESP32 plus a fix (same as wifi wardrive).
# Duration: ~2 min.  SD: required.  GPS: recommended.

stopscan
btwardrive -c
DELAY 120000
stopscan
