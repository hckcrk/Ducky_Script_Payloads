# recon_scan_aps -- discover nearby WiFi access points, list them, save to SD
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Flow: clear old APs -> scan ~25 s -> stop -> print list -> save AP list to SD card
# Duration: ~30 s.  SD: required for `save`.  GPS: not required.
# Tip: note the AP indices printed by `list -a`; other payloads select by index.

stopscan
clearlist -a
scanap
DELAY 25000
stopscan
list -a
save -a
