# beacon_clone_aps -- clone the SSIDs of scanned APs and beacon them back out
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Flow: scan APs -> attack -t beacon -a (clones the scanned AP list as beacons)
# Duration: ~55 s.  SD: not required.  GPS: not required.
# ACTIVE TRANSMISSION -- authorized testing only.

stopscan
clearlist -a
scanap
DELAY 25000
stopscan
list -a
attack -t beacon -a
DELAY 30000
stopscan
