# wardrive_wifi -- WiFi wardrive, logging APs (with GPS coords) to the SD card
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Writes a wardrive CSV to SD. Coordinates are only recorded when GPS is routed to
# the CLI ESP32 (set the GPS DIP switch on the back of the Dual board to the CLI chip)
# AND a satellite fix has been acquired. Without a fix it still logs, but with no coords.
# Duration: ~2 min (increase DELAY for longer drives).  SD: required.  GPS: recommended.

stopscan
wardrive
DELAY 120000
stopscan
