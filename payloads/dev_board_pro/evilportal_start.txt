# evilportal_start -- serve a captive portal page from the SD card
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Prereq: put your portal HTML on the board's SD card first, then set the filename below.
#   evilportal sethtml <file>  -> selects the HTML page to serve
#   evilportal -c start        -> starts the portal AP + web server
# EDIT: replace `portal.html` with your file name as it exists on the SD card.
# Duration: runs ~2 min then stops (raise DELAY to keep it up longer).  SD: required.
# ACTIVE -- authorized testing / demos only.

stopscan
evilportal sethtml portal.html
DELAY 1000
evilportal -c start
DELAY 120000
stopscan
