# recon_full_survey -- APs + stations in one pass, saved to SD
# Target: AWOK Dual Touch V3, orange (CLI) port, esp32_marauder_dev_board_pro.bin, v1.13.0
# Flow: scanap -> scansta -> list both -> save AP list to SD
# Duration: ~60 s.  SD: required for `save`.  GPS: not required.

stopscan
clearlist -a
scanap
DELAY 20000
stopscan
scansta
DELAY 30000
stopscan
list -a
list -c
save -a
